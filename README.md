# Cookie Clicker RP v0.1

Version jouable optimisée pour tablette GTA RP.

## Installation GitHub Pages
1. Crée un dépôt GitHub.
2. Envoie `index.html`, `style.css`, `game.js` à la racine.
3. Va dans Settings > Pages.
4. Source : main branch / root.
5. Utilise le lien GitHub Pages dans ta tablette.

## Fonctionnalités
- Cookie Clicker jouable
- Boutique optimisée
- Achat x1/x10/x100/MAX
- Améliorations
- Succès
- Sauvegarde locale automatique
- Export/import sauvegarde
- Gain hors ligne max 8h
- Son de clic généré en JavaScript
- Volume réglable
